# tsenpeed.github.io

Static GitHub Pages portfolio for Tsen.

## Contents

- `index.html`: semantic single-page portfolio.
- `styles.css`: responsive visual system, layout, motion, and accessibility states.

## Sections

- introduction based on the profile README
- about/current learning
- public project links
- technologies and skills
- GitHub contact links

## Publish

Push this folder to the `main` branch of `https://github.com/tsenpeed/tsenpeed.github.io`.
GitHub Pages will serve it at `https://tsenpeed.github.io`.
